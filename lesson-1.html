This is a paragraph element! These are sortof the generic text elements. You will want to use these for most text that isn't a header or something fancy like that.

See how annoying that long paragraph element is? Code usually doesn't "word-wrap" in code editors, but you can adjust that in the settings. I prefer to do this though, because HTML doesn't care about line breaks in our code, it only cares about the linebreaks we specifically tell it to do! So this formatting won't be visible to the user, but it makes our code easier to read. You can do word-wrap in your code editor's settings though, it doesn't matter, it's up to you!

See where I placed the p tag on this element and the above element? HTML doesn't care about whitespace like line breaks and indents! For elements with a lot of content, it can be easier to read the paragraph without the annoying HTML tags cluttering everything up.

Some other cool HTML elements to know about are the strong element, which bolds the text. The emphasis element, which italicizes the text. There are two elements that do this exact same thing, for bold text and for italicized text. What's the difference?
strong and em carry meaning beyong just the appearance of the text. Screen readers used by blind people will take this into account, and the electronic voice will read it like a human would! Pretty cool!
It's pretty rare that you would want to bold text but not let blind people know about it, so you should usually use strong and em. The i element is often used for font icons, like those from font-awesome.

You've heard of "links" AKA "hyperlinks" on the internet. Well, here's how you make them! You use an "anchor" element, like this: Click here to go to Google.com!

This is an img element, an IMAGE element (I bet you never would've guessed..) the src is the location of the image, which is either another place on the internet or a place on your web server. The alt is text that describes the image. This text is used to describe to blind people what the picture contains, so you want to be very literal with your alt text.

img tags are "self closing", because they don't have any content inside of the tag, like a p element does.

The last thing I'll show you in this lesson is the span element! This element is cool because it doesn't disrupt the flow of the text! You can use this to do all sorts of cool things, but it doesn't do anything on its own.
